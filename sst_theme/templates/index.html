    Need help with a C++ programming assignment?

    C++ can be hard. There are so many different concepts to learn, and many aren’t even related to coding!

    • How to solve a problem
    • How to test that your solution works
    • Setting up a coding environment (such as Visual Studio, Eclipse, Codeblocks, or others)
    • C++ syntax
    • Code concepts
    • Debugging code
    • And much more

    All of those concepts can have several more related items. For example, let’s take the code concepts bullet…

    • if, if-else, if-else if statements
    • Variable types (int, bool, double, float, char, etc.)
    • Functions
    • Loops (while/for)
    • Classes
    • Polymorphism
    • Data structures
    • And much more

    When I was in college learning C++, I can remember how frustrating it was to just start programming. It seemed like magic. What on earth does all this code do? Once I finally realized how to build and run the code, how could I make it work? Why did the stupid compiler keep screaming at me? Oh my gosh, WHY DOESN’T THIS WORK??? JUST WORK!!!! (Several thoughts of chunking the computer out the window crept into my mind.)

    There are several things I did to get through the class. I read sections and chapters of my textbook several times. I kept working on the C++ assignments in the lab we had… sometimes for a really long time to make sure I understood what I was doing. You must understand, learning programming is a step-by-step process. There will be periods of a lot of frustration followed by those “eureka!” moments. That feeling of finally getting something right is one of the best feelings in the world.
